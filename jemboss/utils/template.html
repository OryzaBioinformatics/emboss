Jemboss is being developed by the EMBOSS team. The software is part of the EMBOSS distribution.

Instructions to Start Using Jemboss

  • When you use Jemboss for the first time, you need to have Java Web Start (JWS). JWS is bundled with the Java 2 SDK and Java Runtime Environment (JRE) version 1.4 (or later). It allows you to launch Java applications, such as Jemboss, from the web.
  • Click the button to launch Jemboss



    It will spend some time retrieving files.
    A desktop shortcut icon will be created. Subsequent launches of Jemboss can be executed via the shortcut without the need to return to the Jemboss JWS web page. Subsequent launches will be faster as the Jemboss application will be cached in your system.
