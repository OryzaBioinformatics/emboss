Jemboss is an interface to EMBOSS.  It was developed at the HGMP-RC, UK and is now supported by the EMBOSS team at the European Bioinformatics Institute and the original author Tim Carver at the Wellcome Trust Sanger Institute. Jemboss can be run in one of two modes. The interface can be run with a local installation, this is the stand-alone mode. Alternatively, it may be run using a remote server, i.e. in a client-server mode.
