pepwindowall

 

Function

Displays protein hydropathy of a set of sequences

Description

pepwindowall produces a set of superimposed Kyte & Doolittle hydropathy plots from an aligned set of protein sequences.

The result is the same as running pepwindow on a set of proteins with aligning gaps and superimposing the plots.

It is useful for visualising the average hydropathy and its variability along the alignment.

Usage

Here is a sample session with pepwindowall 


% pepwindowall ../../data/globins.msf -gxtitle="Base Number" -gytitle="hydropathy" 
Displays protein hydropathy of a set of sequences
Graph type [x11]: cps
