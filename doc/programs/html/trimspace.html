trimspace

 

Function

Remove extra whitespace from an ASCII text file

Description

Replaces all white space with a single space. nospace removes all whitespace from the input sequence.

Usage

Here is a sample session with trimspace 


% trimspace 
Remove extra whitespace from an ASCII text file
