psiphi

 

Function

Calculates phi and psi torsion angles from cleaned EMBOSS-style protein co-ordinate file

Description

Calculates the psi and phi torsion angles around the alpha carbons (CA) in (a specified stretch of) a specified chain of a protein structure from the co-ordinates of the mainchain atoms in the two planes around it
