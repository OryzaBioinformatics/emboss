EMBOSS: wordcount

Program wordcount

Function

Counts words of a specified size in a DNA sequence

Description 

Displays all the words of the specified length with the number of
times it occurs.

Usage

Here is a sample session with wordcount. 
% wordcount embl:rnu68037 -wordsize=3
