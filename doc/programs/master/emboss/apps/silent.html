silent

Function

Description

silent does a scan of a nucleic acid sequence for silent mutation restriction enzyme sites.

silent finds positions in a sequence where a point mutation could be made to introduce a specified restriction enzyme recognition site without changing the translation.

Several restriction enzyme can be specified.

Usage

Command line arguments

Input file format

silent reads a nucleic acid sequence USA.

Output file format

Data files

Notes

None.

References

Diagnostic Error Messages

None.

Exit status

It always exits with status 0.

Known bugs

None.

silent does the opposite to recode. silent finds sites where a restriction enzyme site can be introduced without changing the translation in frame 1 of the sequence. recode finds sites where a restriction enzyme site can be removed without changing the translation in frame 1 of the sequence.
