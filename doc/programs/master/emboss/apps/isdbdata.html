isdbdata

Function

Description

isdbdata checks whether a supplied string is the name of a known data type or data identifier according to the EDAM ontology. An informative message is written to screen. Optionally it will write to file the EDAM term statement with a name that matches the supplied string.

Only the EDAM 'data' branch (data type or identifier) is checked. The EDAM ontology in the EMBOSS data file edam.obo is used.

Usage

Command line arguments

Input file format

Output file format

If an output file is specified, the entire OBO term statement from the EDAM ontology will be written to file in OBO plain text format as used by the EMBOSS data file edam.obo and explained at http://www.geneontology.org/GO.format.obo-1_2.shtml.

Data files

The EDAM ontology (edam.obo) must be installed in the EMBOSS data directory and indexed. This is done automatically for typical installations.
