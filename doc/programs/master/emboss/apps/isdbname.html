isdbname

Function

Description

isdbname checks whether a supplied string is a known database name according to the catalogue of public databases in the EMBOSS data file db.dat. An informative message is written to screen. Optionally, if a match is found, information on the database is written to file.

Usage

Command line arguments

Input file format

Output file format

The output file (if specified) is an excerpt of and in the same format as the EMBOSS data file db.dat. The format of the file is explained in comments at the top of that file.

Data files

The EMBOSS database information file (db.dat) must be installed in the EMBOSS data directory and indexed. This is done automatically for typical installations.
