aligncopypair

Function

Description

aligncopypair reads a set of aligned sequences and reports the alignment one pair ar a time using any of the standard EMBOSS alignment formats.

Usage

Command line arguments

Input file format

aligncopypair reads any normal sequence USAs.

Output file format

Data files

For protein sequences EBLOSUM62 is used for the substitution matrix. For nucleotide sequence, EDNAFULL is used.
