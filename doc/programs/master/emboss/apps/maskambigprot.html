maskambigprot

Function

Description

maskambigprot replaces any ambiguity codes in the input sequence with 'X'.

Usage

Command line arguments

Input file format

maskambigprot reads any normal sequence USAs.

Output file format

maskambigprot rewrites the input sequence.
