texttoxml

Function

Description

texttoxml converts a plain text file to an XML format. Sequence, sequence feature and sequence alignment files may be converted.

Usage

Command line arguments

Input file format

See the EMBOSS documentation (http://emboss.sourceforge.net/) for an explanation and examples of the file formats supported for input.

Output file format

See the EMBOSS documentation (http://emboss.sourceforge.net/) for an explanation and examples of the file formats supported for output.

Data files

Notes

The type (sequence, feature or alignment) of the input file and format of the output file must be specified. Support for different file types and file formats is on-going.
