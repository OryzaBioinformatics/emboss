Function

Description

Usage

   Here is a sample session with histogramtest.

% histogramtest

Command line arguments

   Mandatory qualifiers:
   -sets               integer    Number of sets of data
   -points             integer    Number of data points per set
   -bins               integer    Number of bars
   -graph              xygraph    Graph type

   Optional qualifiers:
   -sidebyside         integer    Display sets next to each other
   -xstart             integer    start value for x axis
   -xend               integer    end value for x axis

   Advanced qualifiers: (none)
   General qualifiers:
  -help                bool       report command line options. More
                                  information on associated and general
                                  qualifiers can be found with -help -verbose
   

   Mandatory qualifiers Allowed values Default
   -sets Number of sets of data Integer from 1 to 10 1
   -points Number of data points per set Integer from 1 to 200 10
   -bins Number of bars Integer 1 or more $(points)
   -graph Graph type EMBOSS has a list of known devices, including
   postscript, ps, hpgl, hp7470, hp7580, meta, colourps, cps, xwindows,
   x11, tektronics, tekt, tek4107t, tek, none, null, text, data, xterm,
   png EMBOSS_GRAPHICS value, or x11
   Optional qualifiers Allowed values Default
   -sidebyside Display sets next to each other Integer from 0 to 1 1
   -xstart start value for x axis Any integer value 0
   -xend end value for x axis Any integer value Start value plus number
   of data points
   Advanced qualifiers Allowed values Default
   (none)
