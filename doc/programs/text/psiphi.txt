
                                  psiphi 



Function

   Calculates phi and psi torsion angles from cleaned EMBOSS-style
   protein co-ordinate file

Description

   Calculates the psi and phi torsion angles around the alpha carbons
   (CA) in (a specified stretch of) a specified chain of a protein
   structure from the co-ordinates of the mainchain atoms in the two
   planes around it

   Standard (Mandatory) qualifiers:
  [-infile]            infile     Clean coordinate file
   -chainnumber        integer    Number of the chain for which torsion angles
                                  should be calculated
   -startresiduenumber integer    First residue in chain for which torsion
                                  angles should be calculated
   -finishresiduenumber integer    Last residue in chain for which torsion
                                  angles should be calculated (1 = last

  [-outfile]           report     Output report file name

   -auto               boolean    Turn off prompts
