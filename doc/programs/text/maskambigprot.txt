Function

   Masks all ambiguity characters in protein sequences with X

Description

   maskambigprot replaces any ambiguity codes in the input sequence with
   'X'.

Usage

   Here is a sample session with maskambigprot


% maskambigprot
Masks all ambiguity characters in protein sequences with X
Input (gapped) protein sequence(s): ambigprot.fasta
output sequence(s) [ambigprot.fasta]:

Input file format

   maskambigprot reads any normal sequence USAs.

  Input files for usage example

  File: ambigprot.fasta

>AMBIGPROT
ABCDEFGHIJKLMNOPQRSTUVWXYZ*
abcdefghijklmnopqrstuvwxyz*

Output file format

   maskambigprot rewrites the input sequence.

  Output files for usage example

  File: ambigprot.fasta

>AMBIGPROT
AXCDEFGHIXKLMNXPQRSTXVWXYXXaxcdefghixklmnxpqrstxvwxyxX
